用hugo搭建个人博客

Jul 26, 2019
1 min read

一. 安装hugo

linux： >sudo apt-get install hugo

windows: 可用scoop安装 在powershell中输入： > iex (new-object net.webclient).downloadstring(‘https://get.scoop.sh')

二.使用hugo创建博客

hugo new site myblog //myblog为项目名,博客里的每个页面存储在myblog/content/post/

三.添加内容

hugo new …/blog.md //在content目录下执行

四.运行调试

hugo server –t m10c –buildDrafts

五.主题下载

https://themes.gohugo.io 主题页面里有下载教程

六.启动博客

hugo server -t m10c –buildDrafts // m10c为下载的主题

七.发布博客到网络(github)

  1. 创建新的repository(名字必须为 用户名.github.io)
  2. 关联项目到远端地址: >hugo –theme=m10c –baseUrl=“https://用户名.github.io/” –buildDrafts

会在项目中生成public文件夹 3. 将public推到repository中:

cd public

git init

git add .

git commit -m "my blog"

git remote add origin https

git push -u origin master

八.查看博客

地址为:https://用户名.github.io/
